# Виселица (Hangman) 🎮

Консольная игра "Виселица" на Java. Классическая игра в угадывание слов с пошаговой отрисовкой виселицы.
```
 _______________
      |/      |
      |       O
      |      /|\
      |      / \
      |
      |
  ____|____
```
## Как играть 🎯

1. Программа загадывает случайное слово из словаря
2. Вы по очереди вводите буквы русского алфавита
3. Если буква есть в слове - она открывается
4. Если буквы нет - увеличивается счетчик ошибок и рисуется виселица
5. Игра продолжается до победы (угадано всё слово) или поражения (6 ошибок)

## Особенности ✨

- Поддержка буквы 'ё'
- Красивая ASCII-графика виселицы
- Сортировка использованных букв по алфавиту
- Проверка повторного ввода букв
- Словарь слов в текстовом файле

## Структура проекта 📁
```plaintext
src/main/java/org/example/
├── Game.java # Основная логика игры
├── Word.java # Работа со словом
├── ConsoleInput.java # Ввод с консоли
├── UsedLetters.java # Хранение использованных букв
├── Dictionary.java # Загрузка словаря
├── GallowRenderer.java # Отрисовка виселицы
└── GameStatus.java # Состояния игры
src/main/resources/
└── dictionary.txt # Словарь слов
```
